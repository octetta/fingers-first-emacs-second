## Week 5: Days 29–35

### Typing Practice (10 min/day)

**Bottom row:** z, x, c, v, b (left), n, m, comma, period, slash (right)

The bottom row is hardest because fingers must curl *down* rather than reach *up* or stay level. The left pinky reaching for `z` is particularly awkward at first.

**Days 29–31:** Add `n`, `m`, `c`, `v`
```
man can van mine vine came name
cave maven civic mine canine
```

**Days 32–35:** Add `b`, comma, period; then `z`, `x`
```
combine combine member combine bomb bomb
box fox vex mix exit exist axis
zinc zinc zinc freeze fizz buzz jazz
```

Focus on the left pinky for `z` — it needs conditioning. The comma and period are right-hand ring and middle finger reaching down.

### Emacs Practice (10 min/day)

**Days 29–31: Page navigation**

| Command | Keys | What it does |
|---------|------|-------------|
| Page down | `C-v` | Scroll down one screen |
| Page up | `M-v` | Scroll up one screen |
| Beginning of buffer | `M-<` | Jump to very start of file |
| End of buffer | `M->` | Jump to very end of file |
| Center on cursor | `C-l` | Redraw screen centered on cursor |
| Go to line | `M-g g` | Jump to a specific line number |

**Days 32–35: Dired — the file manager**

Dired is Emacs's built-in directory editor. It's powerful and worth learning early.

```
C-x d              → open Dired (prompts for directory)
C-x C-f <dir>/     → open directory in Dired (include trailing slash)
```

**Inside Dired:**

| Key | What it does |
|-----|-------------|
| `n` / `p` | Move to next/previous file |
| `<RET>` | Open file or directory |
| `^` | Go to parent directory |
| `d` | Mark file for deletion |
| `x` | Execute deletions |
| `R` | Rename/move file |
| `C` | Copy file |
| `+` | Create new directory |
| `!` | Run shell command on file |
| `q` | Quit Dired |
| `g` | Refresh the listing |
| `m` | Mark a file |
| `u` | Unmark |
| `A` | Search all marked files (regexp) |
| `Q` | Query-replace in marked files |

**Exercise:** Use Dired to navigate to a C project directory. Open the `.c` and `.h` files you need, then use `C-x b` to switch between them. Never use a file manager GUI again.

## Week 6: Days 36–42

### Typing Practice (10 min/day)

**Full alphabet consolidation.** You've seen all the keys — now integrate them.

Use [typing.com](https://www.typing.com) or [keybr.com](https://www.keybr.com) on mixed-text mode. Spend these sessions on words that mix all rows:

```
complex mixture bracket clutter various
function pointer structure variable const
```

**Days 36–39:** Speed drill on common English 200-word list  
**Days 40–42:** Type real C keywords and identifiers

```
int long char void struct union enum
return while for if else do switch case
break continue static extern inline const
```

These are your daily vocabulary. Get them automatic.

### Emacs Practice (10 min/day)

**Days 36–38: Frames**

In Emacs, a *frame* is what most people call a window (a separate OS-level window). You can have multiple frames showing different buffers.

| Command | Keys | What it does |
|---------|------|-------------|
| New frame | `C-x 5 2` | Open a new frame |
| Other frame | `C-x 5 o` | Switch to another frame |
| Close frame | `C-x 5 0` | Close current frame |
| Find file in new frame | `C-x 5 f` | Open file in new frame |

**Days 39–42: Registers — Emacs's bookmarks**

Registers let you save positions, text, or rectangle data to single-letter slots.

| Command | Keys | What it does |
|---------|------|-------------|
| Save position | `C-x r <space> <letter>` | Bookmark current position |
| Jump to position | `C-x r j <letter>` | Jump to bookmarked position |
| Save text to register | `C-x r s <letter>` | Copy region to register |
| Insert register | `C-x r i <letter>` | Insert register content |

**Exercise:** In a large C file, mark the main() function with `C-x r <space> m` and the top of the file with `C-x r <space> t`. Practice jumping between them with `C-x r j m` and `C-x r j t`.

## Finger Conditioning Note

By week 6 your weakest fingers are probably your left pinky (Ctrl, `a`, `z`) and your right pinky (`;`, `'`, `/`). This is normal. The Emacs keybindings you're drilling are actively conditioning these fingers. You may notice fatigue in the left pinky particularly — this is fine, but take breaks if it becomes pain. Pain is a stop signal; fatigue is not.
